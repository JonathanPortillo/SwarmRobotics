Authors: Varun Jhunjhunwalla, Shreyas Agarwa, Jonathan Portillox

  • Case 1: If the ’traffic graph’ is acylic and there exists some head agent \(a_i\) that has not reached it’s goal. By definition, this agent has a free path and can reduce \(J_{1}\) by 1.

  • Case 2: If the graph is cyclic. It can be shown by contradiction, that agents will always be able to swap goals in the opposite direction of the cycle without the need to move with non-zero probability. (TODO: refer to random part of psudecode).

  • Case 3: The graph is acyclic and all head agents have reached a goal. This case can essentially be reduced to one of the above cases by swapping goals with a head agent.
